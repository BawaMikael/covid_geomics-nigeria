COVID-19 Genomics Analysis – Northern Nigeria

Overview:
This project analyzes SARS-CoV-2 sequences from Bauchi, Kaduna, and Kano using Ion Torrent data.

Methods:
  Quality control: FastQC
  Variant calling: iVar
  Lineage assignment: Pangolin
  Phylogenetics: IQ-TREE

Data:
Sequences available on GISAID.
